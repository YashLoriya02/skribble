## Skribble Game

### This is the code of game like skribbl, it is built using node.js (express.js) + vite (react.js) + typescript and websocket for RTC.

### Client
- `cd client`
- `npm i`
- `npm run dev`

### Server
- `cd client`
- `npm i`
- `npm run dev`

### Remaining
- Adding more words
- Player removal on tab close
